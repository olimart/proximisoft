Features

Laborder is offered as a hosted solution in the cloud or installed on premise.

Product Inventory

  • Manage any type of products/goods
    • Perishable
    • Radioctive/Hazardous
  • Manage Quantity On Hand (QOH) across multiple warehouses
  • Manage product visiblity and storages
  • Product Catalog
    • Save to favourites
    • Compare products
  • Barcode support

Requisition

  • Submit requistion:
    • Free-form requisition
    • Product in inventory
  • Approval process between buyer (lab technician for instance) and account delegate (Principle Investigator for instance)
  • Workflow process between buyer and procurement services
  • Assign requisition
  • Manage requisition lifecycle and status
  • Handle multiple carts

Receiving

  • Package reception and replenishment
  • Track package reception
  • Track package pickup

System Administration

  • Users Management
    • LDAP Integration (Active Directory - on-premise only)
    • Roles and permissions
  • Manage suppliers
    • Manage promotions
  • Manage storages
  • Manage carriers
  • Manage services and users

Client

  • Personnalized dahboard for all users
  • Access convenient resources at a fingertip
  • Orders overview
    • Ability to re-order
    • Powerful search

And More

  • Web-based Solution
  • Web API
  • Reports
  • Integrate with your systems & workflow
  • Bilingual Application (English & French)
  • Roadmap Available
  • Custom Development Service offered to tailor your specific needs
